# MY ROOM (Room Escape Game)
- DEMO
⭐️ 追加予定

- What
  - シンプルな2D脱出ゲームです。
  - クリアの想定所要時間は5分です。
- How to play
  - 色々な場所を調べてナゾを解き、部屋から脱出できればゲームクリアとなります。

  - 子供の頃にゲームが大好きで、ずっと自分で作ってみたいと思っていた為開発を決めました。
